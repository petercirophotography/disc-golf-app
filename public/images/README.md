# Course Map Images

Place course map screenshots here with the following names:

- `redhawk-map.jpg` - Another Round Lula Red Hawk course map
- `og-map.jpg` - Another Round Lula OG Course map
- `perkerson-map.jpg` - Perkerson Park course map
- `whiteoak-map.jpg` - White Oak Park course map
- `oregon-map.jpg` - Oregon Park course map
- `redan-map.jpg` - Redan Park course map

## How to get course maps:

1. Go to https://udisc.com
2. Search for the course name
3. Take a screenshot of the course map view
4. Save it with the corresponding filename above
5. Place it in this folder

The maps will automatically appear as thumbnails on the course cards.
